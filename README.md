# Nick Aurino's Portfolio

My personal portfolio site, where I showcase my projects, background, and the things I build. I'm a CS graduate who makes software inspired by things I wish existed, from iOS apps (Dabble, Cadence) to published research (M4Health).

🌐 **Live:** [nick-aurino.vercel.app](https://nick-aurino.vercel.app)

## Built with

`Next.js` · `React` · `Tailwind CSS` · `TypeScript`, deployed on Vercel.

No animation libraries: the motion and effects are pure CSS keyframes and vanilla JavaScript.

## Run locally

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).
